# Buscador de Autos

## Descripcion
Pagina web que le permite a usuarios buscar auto segun sus caracteristicas deseadas asi como la marca, a√±o, precio, puerta, transmision y color.
Los datos de los autos se obtien de un archivo aparte dentro del mismo proyecto actuando como una API local
 
## Uso
1. Abre el archivo index.html en el navegador
2. usa las ociones de filtrado para hacer una busqueda segun tus preferencias
3. Los resultados de la busqueda se actualizaran en tiempo real en funcion de los filtros aplicados

## Estructura de archivos
- index.html: la estructura principal de la pagina
- app.css: estilos personalizados
- db.js: archivo con los datos de los autos(Actua como una API local)
- app.js: Archivo javascript que maneja la funcionalidad de la pagina

## Tecnologias
- HTML
- CSS
  - Skeleton.css
  - Normalize.css
  - App.css(estilos personalizados)
- JavaScript
